# 高级预测模型逻辑说明

> 源文件: `probability/calculate_advanced_probability.py`

## 概述

高级预测模型是一个**融合模型**，结合了以下三种核心方法：

1. **攻防效率分析** — 计算主客场的进攻/防守效率
2. **泊松分布** — 基于预期进球数推导全场胜平负概率
3. **半全场状态转移矩阵** — 基于历史半场→全场转移概率

最终通过加权融合（泊松70% + 状态矩阵30%）得到预测结果。

---

## 详细计算流程

### 第1步：计算球队表现数据 (`calculate_team_performance`)

从历史交锋数据中提取每支球队的表现指标：

#### 1.1 全场数据统计

| 指标 | 计算方式 |
|------|---------|
| 总进球数 | 主场进球 + 客场进球 |
| 总失球数 | 主场失球 + 客场失球 |
| 主场进攻 | 主场进球数 / 主场场次 |
| 主场防守 | 主场失球数 / 主场场次 |
| 客场进攻 | 客场进球数 / 客场场次 |
| 客场防守 | 客场失球数 / 客场场次 |

#### 1.2 半全场节奏统计

| 指标 | 计算方式 |
|------|---------|
| 上半场进攻 | 上半场进球数 / 总场次 |
| 上半场防守 | 上半场失球数 / 总场次 |
| 下半场进攻 | (全场进球 - 上半场进球) / 总场次 |
| 下半场防守 | (全场失球 - 上半场失球) / 总场次 |

#### 1.3 Wilson得分修正

使用 **Wilson得分区间** 对小样本的胜/平/负概率进行修正：

```
Wilson中心点 = (p + z²/(2n)) / (1 + z²/n)
```

- `p` = 观察比例（如胜场数/总场次）
- `n` = 总场次
- `z` = 标准正态分布分位数（95%置信水平下 z=1.96）

对胜、平、负三个类别分别计算Wilson修正概率，然后**归一化**确保概率和为1。

---

### 第2步：共同对手实力分析 (`analyze_common_opponents_strength`)

#### 2.1 找出共同对手

从主队和客队的历史比赛记录中，找出两队都交战过的对手。

#### 2.2 计算实力对比分数

对每个共同对手：
- 主队对该对手的表现评分：胜=1.0, 平=0.5, 负=0.0，取平均
- 客队对该对手的表现评分：同上

最终实力对比分数：
```
home_strength_ratio = home_avg / (home_avg + away_avg)
```

- `> 0.5` 表示主队优势
- `= 0.5` 表示势均力敌
- `< 0.5` 表示客队优势

---

### 第3步：计算预期进球数

#### 3.1 基础预期进球

```
home_expected = 主队主场进攻 × 客队客场防守 × 主场优势系数(1.15)
away_expected = 客队客场进攻 × 主队主场防守
```

#### 3.2 共同对手调整

```python
# 调整幅度与共同对手数量相关（对数递减）
base_adjustment = 0.5 × (1 - e^(-n/3))   # n=共同对手数

# 调整方向由实力对比决定
adjustment = base_adjustment × (strength_ratio - 0.5) × 2

# 应用调整
home_expected *= (1 + adjustment)
away_expected *= (1 - adjustment)
```

#### 3.3 预期进球约束 (`constrain_expected_goals`)

对预期进球数施加多重约束：

| 约束类型 | 规则 |
|---------|------|
| 范围约束 | 单队预期进球 ∈ [0.1, 3.5] |
| 比例约束 | 两队进球比 ≤ 3.0 |
| 联赛均值回归 | 20%收缩向联赛均值1.4 |
| 进攻效率约束 | 预期进球/进攻效率 ∈ [0.5, 2.0] |

---

### 第4步：泊松分布计算 (`calculate_poisson_probability`)

使用泊松分布计算比分概率矩阵：

```
P(主队进i球) = e^(-λ_home) × λ_home^i / i!
P(客队进j球) = e^(-λ_away) × λ_away^j / j!
P(比分i:j) = P(主队进i球) × P(客队进j球)
```

最大进球数限制为5球，然后汇总：

- **胜概率** = Σ P(i:j) 其中 i > j
- **平概率** = Σ P(i:j) 其中 i = j
- **负概率** = Σ P(i:j) 其中 i < j

最后归一化确保概率和为1。

---

### 第5步：半全场状态转移矩阵 (`create_half_full_state_matrix`)

#### 5.1 统计方法（优先）

从比赛记录中统计半场→全场的转移概率矩阵（3×3）：

| | 全场胜 | 全场平 | 全场负 |
|---|---|---|---|
| **半场胜** | P(W\|W) | P(D\|W) | P(L\|W) |
| **半场平** | P(W\|D) | P(D\|D) | P(L\|D) |
| **半场负** | P(W\|L) | P(D\|L) | P(L\|L) |

分别计算主队和客队的转移矩阵，然后取平均。

#### 5.2 独立性假设方法（备选）

当没有比赛记录时，使用Wilson修正后的半场和全场结果概率，通过独立性假设估计联合分布：

```
P(半场结果, 全场结果) ≈ P(半场结果) × P(全场结果)
```

然后归一化每行得到条件概率。

默认转移矩阵：

| | 全场胜 | 全场平 | 全场负 |
|---|---|---|---|
| **半场胜** | 0.6 | 0.3 | 0.1 |
| **半场平** | 0.3 | 0.4 | 0.3 |
| **半场负** | 0.1 | 0.3 | 0.6 |

---

### 第6步：基于状态矩阵计算全场概率

```python
# 主队视角：半场概率 × 转移矩阵 → 全场概率
home_full_from_state = home_half_probs @ state_matrix

# 客队视角（需转换为主队视角）
away_full_from_home = [客队负, 客队平, 客队胜]  # 反转

# 结合两队
state_based_probs = (home_full + away_full_from_home) / 2
```

---

### 第7步：加权融合

```python
poisson_weight = 0.7    # 泊松分布权重
state_weight = 0.3      # 状态矩阵权重

combined_home_win = poisson_win × 0.7 + state_win × 0.3
combined_draw     = poisson_draw × 0.7 + state_draw × 0.3
combined_away_win = poisson_loss × 0.7 + state_loss × 0.3

# 归一化
total = combined_home_win + combined_draw + combined_away_win
final_probs = [combined_home_win, combined_draw, combined_away_win] / total
```

---

### 第8步：半场进攻效率微调

```python
# 下半场进攻因子 = 下半场进攻 / 上半场进攻
home_second_half_factor = home_second_half_attack / (home_first_half_attack + 0.001)
away_second_half_factor = away_second_half_attack / (away_first_half_attack + 0.001)

# 微调幅度5%
home_adjustment = 1.0 + (home_second_half_factor - 1.0) × 0.05
away_adjustment = 1.0 + (away_second_half_factor - 1.0) × 0.05

# 应用调整（平局不调整）
final_home_win = combined_home_win × home_adjustment
final_draw     = combined_draw
final_away_win = combined_away_win × away_adjustment

# 再次归一化
```

## 核心函数一览

| 函数 | 作用 |
|------|------|
| `wilson_score` | Wilson得分区间修正小样本概率 |
| `multinomial_wilson` | 多项分布Wilson修正（胜/平/负） |
| `calculate_home_advantage_factor` | 主场优势系数（固定1.15） |
| `calculate_common_opponent_adjustment` | 共同对手调整系数 |
| `constrain_expected_goals` | 预期进球数多重约束 |
| `calculate_team_performance` | 球队攻防效率和半全场节奏 |
| `analyze_common_opponents_strength` | 共同对手实力对比分析 |
| `calculate_poisson_probability` | 泊松分布计算比分概率 |
| `calculate_transition_matrix_from_matches` | 从比赛记录统计转移矩阵 |
| `create_half_full_state_matrix` | 创建半全场状态转移矩阵 |
| `calculate_final_probabilities` | 主函数：融合所有步骤计算最终概率 |

## 与其他模型的对比

| 特性 | 共同对手泊松 | 直接交锋泊松 | 高级预测 |
|------|------------|------------|---------|
| 数据来源 | 共同对手比赛 | 直接交锋记录 | 全部历史比赛 |
| 核心方法 | 泊松分布 | 泊松分布 | 泊松+状态矩阵融合 |
| 半场概率 | ✅ 独立计算 | ✅ 独立计算 | ❌ 通过状态矩阵间接考虑 |
| 时间衰减 | ✅ 指数衰减 | ✅ 指数衰减 | ❌ 无 |
| 主场优势 | 1.10 | 1.10 | 1.15 |
| 平局提升 | ✅ 接近度提升 | ✅ 接近度提升 | ❌ 无 |
| 小样本修正 | ❌ | ✅ 贝叶斯收缩 | ✅ Wilson得分 |
| 预期进球约束 | ❌ | ❌ | ✅ 多重约束 |
