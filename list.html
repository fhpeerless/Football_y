计算方法说明

实力分分析 泊松

基于共同对手数据的泊松模型：
• 时间衰减加权计算攻击力/防守力
• 综合攻防估算预期进球(lambda)
• 泊松分布推导胜平负概率
• 主胜概率最大 → 3
• 平概率最大 → 1
• 客胜概率最大 → 0

高级预测概率 高级

比较主胜、平、客胜概率：
• 主胜概率最大 → 3
• 平概率最大 → 1
• 客胜概率最大 → 0

直接交锋泊松 泊松

基于两队直接交锋记录的泊松模型：
• 时间衰减加权计算主客场攻击力
• 实力接近时平局概率自动提升
• 泊松分布推导胜平负概率
• 主胜概率最大 → 3
• 平概率最大 → 1
• 客胜概率最大 → 0
